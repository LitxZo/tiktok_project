## 目录结构说明
- api：相当于controllor，处理web请求的方法都放在里面。
- cmd：启动项目的相关命令
- conf：系统配置相关
- dao：数据库相关
- global：全局相关
- log：日志
- middleware：中间件
- model：数据库model
- router：路由
- service：服务相关
- utils：工具

## 下载依赖
```
go mod tidy
```

## 运行项目
```
go run main.go
```
